联想G560刷Casaos系统作家庭NAS

我有一台旧联想G560笔记本，因为闲置很久了，所以一直没有动。前几天我把闲置的这台联想G560笔记本电脑利用起来，用来做NAS，而我选择的NAS系统是Casaos。而我为什么选择Casaos系统做NAS，而不是其它，是因为Casaos有其优点：

1.安装简单，部署门槛低 2.为图形化桌面系统 3.内置docker生态 4.对于旧硬件，比如我这台G560做nas十分友好 5.文件管理和共享方便 6.可扩展性强

由于Casaos不是独立系统，所以我们需要在G560上刷一个Linux系统安装，这里我选择 Debian13：

下载好Rufus，插入U盘，准备将刚刚下载好的Debian13镜像刷入U盘，做系统盘：
写入Debian系统，并完成

然后将刚刚刷好的Debian的U盘插入G560笔记本，开机按F12进入BIOS，选择以USB HDD启动后，进入Debian设置直到完成步骤：

语言选择中文，地区为中国，键盘选择中文，由于我插了网线，这里选择enp7s0。这里有二个选项：一个是 wlp6s0，一个是enp7s0,前面是要网络连接选择是wifi，后者是有线连接。当然我选择enp7s0,而且此电脑只支持2.4Ghz的wifi，不怎么稳定。接下来，设置密码，并创建用户。磁盘分区Guided - use entire disk，再All files in one partition，最后Finish partitioning and write changes to disk。接下来选择安装Debian软件，只选择SSH server 和standard system utilities就行了。

安装完成，提示Remove installation media，拔掉U盘，并重启G560。

再次重启G560后：

登录Debian，查看ip:

ip a

再安装Casaos,命令如下：

curl -fsSL https://get.casaos.io | sudo bash

安装完成，可以看到访问ip。

打开192.168.50.191访问地址，设置好Casaos用户名以及密码,登录：

这样就可以看到Casaos界面了，如果觉得G560笔记本自带容量不够多，你还可以买一个移动硬盘，加载到G560 。

但是随之而来的问题是，如果断电，或者此G560休眠了，随机访问地址变了怎么办， 这里可以绑定路由器，我前面说过了，此G560是有线，连接的是路由器，我这台路由器是netgear R8500,由于某些原因，刷了官改梅林华硕固件，流程是：

登录R8500后台》打开内部网络（lan口）》打开dhcp服务器》手动指定IP的dhcp列表并填好

改完后，此后不管是G560断电好还是休眠好，只要以后登录Debian后台。访问Casaos地址就会固定不变，这样以后不管手机还是电脑，都能访问这个Casaos的NAS系统，至于NAS能做些什么，这些就不用我说了吧。
